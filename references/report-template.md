## 表格规范

### 表格样式
- 使用Markdown标准表格格式
- 表头使用粗体
- 数值保留2位小数（百分比除外）
- 评级使用统一标准：优秀/良好/一般、高/中/低、偏高/合理/偏低

### 表格示例
```
| 指标 | 数值 | 说明 |
|------|------|------|
| ROE | 15.23% | 优秀 |
| PE | 25.50 | 合理 |
| 负债率 | 45.00% | 中等 |
```

## 图表使用规范

### 图表类型
1. **股价走势图**：折线图 + 均线
2. **K线图**：蜡烛图 + 成交量
3. **MACD图**：双线图 + 柱状图
4. **RSI图**：折线图 + 超买超卖区域
5. **财务趋势图**：多系列折线图或柱状图
6. **情绪趋势图**：折线图
7. **风险雷达图**：雷达图
8. **多空对比图**：对比柱状图或雷达图

### 图表调用方式
```bash
# 生成所有图表
python scripts/chart_generator.py --symbol 600519.SH --market cn --chart-type all --output-dir ./charts

# 生成单个图表
python scripts/chart_generator.py --symbol 600519.SH --market cn --chart-type price --output-dir ./charts
```
