## 基本面分析师框架

### 核心分析维度
1. **财务健康度**
   - 营收增长率：近3年趋势
   - 净利润率：行业对比
   - ROE（净资产收益率）：是否>15%
   - 负债率：是否<60%
   - 现金流：经营活动现金流是否为正

2. **估值水平**
   - PE（市盈率）：历史分位数
   - PB（市净率）：行业对比
   - PEG：是否<1（合理估值）
   - EV/EBITDA：适用于重资产行业

3. **竞争优势**
   - 品牌护城河
   - 技术壁垒
   - 成本优势
   - 网络效应

4. **增长动力**
   - 产品/服务创新
   - 市场扩张
   - 并购整合
   - 政策红利

### 检查清单
- [ ] 查阅最近3个季度的财报
- [ ] 对比同行业竞争对手的关键指标
- [ ] 识别潜在的危险信号（应收账款激增、存货积压等）
- [ ] 评估管理层的执行力和诚信度
- [ ] 判断是否具有可持续竞争优势

### 输出格式
```
基本面分析结论：
- 财务评分：X/10
- 估值评分：X/10
- 竞争优势评分：X/10
- 综合评分：X/10
- 核心观点：[一句话总结]
- 关键风险：[列出2-3个主要风险点]
```

## 情绪分析师框架

### 情绪评估维度
1. **市场情绪指标**
   - 换手率：活跃度判断
   - 资金流向：净流入/流出
   - 大单交易：机构动向
   - 融资融券余额：杠杆水平

2. **社交媒体情绪**
   - 讨论热度：相关话题热度
   - 正负面情绪比例
   - 关键意见领袖（KOL）观点

3. **投资者行为**
   - 散户vs机构持仓比例
   - 股东人数变化
   - 内部人交易

### 情绪评分标准（1-10分）
- 9-10分：极度乐观（警惕泡沫）
- 7-8分：乐观
- 5-6分：中性
- 3-4分：悲观
- 1-2分：极度悲观（可能存在机会）

### 检查清单
- [ ] 分析近期成交量变化
- [ ] 监控社交媒体讨论热度
- [ ] 识别情绪极端化信号
- [ ] 对比历史情绪周期

### 输出格式
```
情绪分析结论：
- 当前情绪评分：X/10
- 情绪状态：[极度乐观/乐观/中性/悲观/极度悲观]
- 资金流向：[净流入/净流出]，金额：X亿元
- 换手率：X%，处于[高/中/低]水平
- 短期情绪预测：[乐观/谨慎/悲观]
```

## 新闻分析师框架

### 新闻分析维度
1. **公司新闻**
   - 业绩预告/财报发布
   - 重大合同/项目
   - 并购重组
   - 高管变动

2. **行业新闻**
   - 行业政策变化
   - 行业发展趋势
   - 竞争对手动态
   - 技术突破

3. **宏观经济**
   - 货币政策（利率、降准）
   - 财政政策（减税、基建）
   - 国际关系
   - 重大事件（疫情、战争等）

### 影响程度评估
- 高影响：直接影响公司基本面
- 中影响：间接影响行业景气度
- 低影响：短期情绪波动

### 检查清单
- [ ] 梳理最近1个月的重要新闻
- [ ] 区分利好/利空/中性
- [ ] 评估新闻的时效性和持续性
- [ ] 识别可能被过度解读的新闻

### 输出格式
```
新闻分析结论：
- 重大利好：[列出1-2个]
- 重大利空：[列出1-2个]
- 中性事件：[列出1-2个]
- 宏观环境影响：[正面/负面/中性]
- 新闻综合影响评分：X/10
```

## 技术分析师框架

### 核心技术指标

1. **趋势指标**
   - MA（移动平均线）：MA5、MA10、MA20、MA60
   - 多头排列：MA5 > MA10 > MA20 > MA60
   - 空头排列：MA5 < MA10 < MA20 < MA60

2. **动量指标**
   - MACD：金叉（买入信号）、死叉（卖出信号）
   - RSI：>70超买、<30超卖、50为多空分界
   - KDJ：超买超卖信号

3. **波动指标**
   - 布林带：价格触及上轨/下轨
   - ATR：波动率水平

4. **成交量**
   - 量价配合：价涨量增、价跌量缩
   - 量价背离：价涨量缩、价跌量增

### 买卖点判断

**买入信号（满足2个以上）**:
- MACD金叉
- RSI从超卖区回升
- 股价站稳MA20上方
- 成交量放大配合价格上涨
- 多头排列形成

**卖出信号（满足2个以上）**:
- MACD死叉
- RSI进入超买区
- 股价跌破MA20
- 高位放量滞涨
- 空头排列形成

### 精确买卖点位
```
买入点位：
- 第一买点：[价格区间]，理由：[技术依据]
- 第二买点：[价格区间]，理由：[技术依据]
- 止损位：[价格]，理由：[技术依据]

卖出点位：
- 止盈位：[价格]，理由：[技术依据]
- 减仓位：[价格]，理由：[技术依据]
```

### 检查清单
- [ ] 检查均线系统排列
- [ ] 分析MACD金叉/死叉信号
- [ ] 评估RSI是否超买超卖
- [ ] 观察布林带位置
- [ ] 分析成交量配合情况
- [ ] 识别关键支撑位和阻力位

### 输出格式
```
技术分析结论：
- 趋势判断：[上升趋势/下降趋势/震荡]
- 均线系统：[多头排列/空头排列/纠缠]
- MACD：[金叉/死叉/信号不明]，强度：[强/中/弱]
- RSI：[X]，状态：[超买/超卖/正常]
- 布林带：位置：[上轨/中轨/下轨/突破]
- 成交量：[放量/缩量/正常]
- 综合技术评分：X/10
- 操作建议：[强烈买入/买入/持有/卖出/强烈卖出]
```

## 研究员辩论框架

### 辩论流程
1. **看多研究员发言**
   - 列举3-5个核心看涨理由
   - 引用分析师团队的数据支撑
   - 评估上涨空间和目标价位

2. **看空研究员发言**
   - 列举3-5个核心看跌理由
   - 反驳看多研究员的观点
   - 评估下跌风险和潜在损失

3. **辩论总结**
   - 权衡多空双方论点
   - 给出综合判断
   - 明确主要风险和机会

### 辩论输出格式
```
研究员辩论结论：

【看多观点】
1. [观点1]，支撑：[数据]
2. [观点2]，支撑：[数据]
3. [观点3]，支撑：[数据]

【看空观点】
1. [观点1]，支撑：[数据]
2. [观点2]，支撑：[数据]
3. [观点3]，支撑：[数据]

【综合判断】
- 看多权重：X%
- 看空权重：X%
- 核心结论：[一句话总结]
- 关键风险：[最需要关注的1-2个风险]
- 潜在收益：[预估上涨空间]
- 潜在风险：[预估下跌空间]
```
