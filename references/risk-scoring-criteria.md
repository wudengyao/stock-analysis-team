## 评分总则

### 评分原则
1. **多维度综合**：基于基本面、技术面、情绪面、消息面等多个维度综合评估
2. **客观量化**：尽可能使用量化指标，减少主观判断
3. **动态调整**：随着市场环境变化，评分需要及时更新
4. **风险为本**：以风险为核心，高分代表高风险，低分代表低风险

### 评分范围
- **1-3分**：低风险
- **4-6分**：中等风险
- **7-8分**：高风险
- **9-10分**：极高风险

## 评分维度

### 1. 基本面风险（权重30%）
**评估指标**:
- 财务健康度：负债率、现金流、ROE
- 估值水平：PE、PB是否偏离历史均值
- 盈利能力：营收和利润增长率
- 行业地位：市场份额、竞争优势

**评分标准**:
- 财务健康、估值合理、盈利稳定、行业龙头 → 1-3分
- 财务一般、估值偏高、盈利放缓、行业前列 → 4-6分
- 财务较差、估值过高、盈利下滑、行业跟随者 → 7-8分
- 财务恶化、估值泡沫、亏损、行业边缘 → 9-10分

### 2. 技术面风险（权重25%）
**评估指标**:
- 趋势状态：上升/下降/震荡
- 技术指标：MACD、RSI、布林带
- 关键位置：支撑位/阻力位距离
- 波动率：ATR、布林带宽度

**评分标准**:
- 上升趋势、指标健康、远离阻力位、低波动 → 1-3分
- 震荡趋势、指标中性、处于中位、中等波动 → 4-6分
- 下降趋势、指标走弱、接近支撑位、高波动 → 7-8分
- 崩盘趋势、指标超卖、跌破支撑位、极端波动 → 9-10分

### 3. 情绪面风险（权重20%）
**评估指标**:
- 市场情绪评分：1-10分
- 换手率：活跃度
- 资金流向：净流入/流出
- 散户vs机构：持仓比例

**评分标准**:
- 情绪中性（4-6分）、换手率低、资金平衡 → 1-3分
- 情绪乐观（6-7分）、换手率中等、资金小幅流入 → 4-6分
- 情绪过热（8-9分）、换手率高、资金大量流入 → 7-8分
- 情绪疯狂（10分）、换手率异常、资金疯狂涌入 → 9-10分

### 4. 消息面风险（权重15%）
**评估指标**:
- 重大利好/利空事件
- 行业政策变化
- 宏观环境影响
- 公司治理风险

**评分标准**:
- 无重大事件、政策中性、宏观稳定 → 1-3分
- 中性偏多/偏空、政策平稳、宏观一般 → 4-6分
- 重大利空、政策收紧、宏观转差 → 7-8分
- 暴雷风险、政策重大打击、宏观危机 → 9-10分

### 5. 市场环境风险（权重10%）
**评估指标**:
- 大盘阶段（进攻/均衡/防守）
- 行业景气度
- 系统性风险
- 流动性环境

**评分标准**:
- 大盘进攻、行业景气、流动性充裕 → 1-3分
- 大盘均衡、行业平稳、流动性正常 → 4-6分
- 大盘防守、行业低迷、流动性偏紧 → 7-8分
- 大盘崩盘、行业衰退、流动性危机 → 9-10分

## 评分标准

### 计算公式
```
风险评分 = 基本面风险 × 30% + 技术面风险 × 25% + 情绪面风险 × 20% + 消息面风险 × 15% + 市场环境风险 × 10%
```

### 评分调整因子
- **+1分**：出现以下情况时上调1分
  - 连续3个季度净利润下滑
  - 重大并购重组失败
  - 监管处罚或调查
  - 关键人物离职

- **+2分**：出现以下情况时上调2分
  - 财务造假嫌疑
  - 债务违约风险
  - 行业遭遇重大政策打击
  - 公司面临退市风险

- **-1分**：出现以下情况时下调1分
  - 超预期业绩增长
  - 重大技术突破
  - 政策红利
  - 行业景气度显著提升

## 等级定义

### 1-3分：低风险
**特征**:
- 基本面扎实，财务健康
- 技术面良好，趋势向上
- 情绪理性，估值合理
- 无重大风险事件
- 市场环境友好

**操作建议**:
- 建议积极布局
- 仓位可提升至70%-90%
- 适合长期持有
- 止损标准：8%-10%

**代表案例**:
- 行业龙头、业绩稳定增长
- 估值处于历史低位
- 技术面形成多头排列

### 4-6分：中等风险
**特征**:
- 基本面一般，存在瑕疵
- 技术面震荡，方向不明
- 情绪中性和，估值合理偏高
- 偶有利空消息
- 市场环境平稳

**操作建议**:
- 建议均衡配置
- 仓位控制在40%-60%
- 适合波段操作
- 止损标准：6%-8%

**代表案例**:
- 行业中游公司
- 估值处于中位
- 技术面震荡整理

### 7-8分：高风险
**特征**:
- 基本面恶化，盈利下滑
- 技术面走弱，趋势向下
- 情绪过热或恐慌
- 重大利空消息
- 市场环境恶劣

**操作建议**:
- 建议谨慎参与
- 仓位控制在10%-30%
- 适合快进快出
- 止损标准：3%-5%

**代表案例**:
- 亏损企业或ST股票
- 估值泡沫严重
- 技术面空头排列

### 9-10分：极高风险
**特征**:
- 基本面崩溃，濒临破产
- 技术面崩盘，自由落体
- 情绪极度恐慌
- 暴雷风险极高
- 市场环境危机

**操作建议**:
- 建议回避
- 仓位控制在0%-10%
- 严禁参与或仅限极少量试水
- 止损标准：2%-3%

**代表案例**:
- 退市风险股
- 财务造假嫌疑
- 债务违约风险

## 评分示例

### 示例1：优质白马股
```
股票：贵州茅台 600519.SH

基本面风险：2分（财务优秀、行业龙头）
技术面风险：3分（上升趋势、指标健康）
情绪面风险：4分（情绪偏热、换手率中等）
消息面风险：2分（无重大利空）
市场环境风险：3分（大盘均衡）

综合评分 = 2×0.3 + 3×0.25 + 4×0.2 + 2×0.15 + 3×0.1 = 0.6 + 0.75 + 0.8 + 0.3 + 0.3 = 2.75分

调整：无
最终风险评分：3分（低风险）

操作建议：积极布局，仓位70%-90%
```

### 示例2：题材炒作股
```
股票：某热门概念股

基本面风险：7分（盈利差、无实质业务）
技术面风险：6分（高位震荡、指标背离）
情绪面风险：9分（情绪疯狂、换手率异常）
消息面风险：5分（题材炒作、监管风险）
市场环境风险：4分（大盘均衡）

综合评分 = 7×0.3 + 6×0.25 + 9×0.2 + 5×0.15 + 4×0.1 = 2.1 + 1.5 + 1.8 + 0.75 + 0.4 = 6.55分

调整：+1分（连续3个季度亏损）
最终风险评分：8分（高风险）

操作建议：谨慎参与，仓位10%-30%，快进快出
```

### 示例3：ST风险股
```
股票：某ST股票

基本面风险：9分（财务恶化、濒临退市）
技术面风险：8分（连续跌停、自由落体）
情绪面风险：10分（极度恐慌、流动性枯竭）
消息面风险：9分（退市风险）
市场环境风险：7分（大盘防守）

综合评分 = 9×0.3 + 8×0.25 + 10×0.2 + 9×0.15 + 7×0.1 = 2.7 + 2.0 + 2.0 + 1.35 + 0.7 = 8.75分

调整：+2分（面临退市风险）
最终风险评分：10分（极高风险）

操作建议：坚决回避
```
