## A股三段式复盘策略

### 策略概述
三段式复盘策略是根据市场周期将投资决策分为三个阶段：进攻、均衡、防守。通过识别当前市场所处阶段，制定相应的仓位和操作策略。

### 阶段识别标准

#### 1. 进攻阶段（Risk-On）
**市场特征**:
- 大盘指数处于上升通道，突破关键阻力位
- 成交量持续放大
- 大部分板块上涨，市场情绪乐观
- MACD金叉，RSI>50但未超买
- 北向资金持续净流入

**操作策略**:
- 建议仓位：70%-90%
- 板块选择：进攻性板块（科技、新能源、军工）
- 操作节奏：积极布局，追涨杀跌
- 止损标准：5%-8%

**信号判断**（满足3个以上）:
- [ ] 上证指数站上MA20且MA5>MA10>MA20
- [ ] 成交量较前期放大20%以上
- [ ] 涨停家数>跌停家数
- [ ] 北向资金连续3日净流入
- [ ] 市场情绪评分>6分

#### 2. 均衡阶段（Neutral）
**市场特征**:
- 大盘指数震荡整理，方向不明
- 成交量温和
- 板块轮动明显，缺乏主线
- 技术指标中性（RSI在40-60之间）
- 资金流向平衡

**操作策略**:
- 建议仓位：40%-60%
- 板块选择：均衡配置（消费+医药+制造）
- 操作节奏：波段操作，高抛低吸
- 止损标准：8%-10%

**信号判断**（满足3个以上）:
- [ ] 指数在MA20上下震荡
- [ ] 成交量与前周基本持平
- [ ] 涨跌家数相当
- [ ] 北向资金小幅波动
- [ ] 市场情绪评分4-6分

#### 3. 防守阶段（Risk-Off）
**市场特征**:
- 大盘指数处于下降趋势，跌破关键支撑位
- 成交量萎缩或异常放大（恐慌性抛售）
- 大部分板块下跌，市场情绪悲观
- MACD死叉，RSI<40
- 北向资金持续净流出

**操作策略**:
- 建议仓位：10%-30%
- 板块选择：防御性板块（银行、公用事业、必需消费）
- 操作节奏：减仓观望，等待企稳
- 止损标准：3%-5%

**信号判断**（满足3个以上）:
- [ ] 上证指数跌破MA20且MA5<MA10<MA20
- [ ] 成交量较前期萎缩30%以上
- [ ] 跌停家数>涨停家数
- [ ] 北向资金连续3日净流出
- [ ] 市场情绪评分<4分

## 美股 Regime Strategy

### 策略概述
Regime Strategy（制度策略）是根据宏观经济环境将市场状态分为三种制度（Regime）：Risk-On（风险偏好）、Neutral（中性）、Risk-Off（风险厌恶）。不同制度下，资产表现差异显著，需要采用不同的投资策略。

### 制度识别标准

#### 1. Risk-On 制度
**市场特征**:
- 经济数据强劲，GDP增长>3%
- 通胀温和，利率稳定或下降
- 企业盈利增长强劲
- 标普500指数创新高或接近高点
- VIX指数<20（低波动）
- 高收益债券利差收窄

**操作策略**:
- 建议仓位：80%-100%
- 板块选择：成长股（科技、生物医药）、小盘股、新兴市场
- 行业偏好：科技、可选消费、工业
- 避免板块：公用事业、必需消费

**信号判断**（满足3个以上）:
- [ ] GDP季度增长率>3%
- [ ] CPI通胀<3%
- [ ] 标普500指数站上200日均线
- [ ] VIX<20
- [ ] 高收益债券利差<5%

#### 2. Neutral 制度
**市场特征**:
- 经济增长平稳，GDP增长1%-3%
- 通胀稳定
- 企业盈利增速放缓
- 标普500指数在历史高点10%范围内震荡
- VIX指数在20-30之间（中等波动）
- 市场缺乏明确方向

**操作策略**:
- 建议仓位：50%-70%
- 板块选择：均衡配置（成长+价值）
- 行业偏好：医疗健康、金融、消费
- 操作节奏：选股为主，降低对beta的依赖

**信号判断**（满足3个以上）:
- [ ] GDP季度增长率1%-3%
- [ ] CPI通胀2%-4%
- [ ] 标普500指数在200日均线附近震荡
- [ ] VIX在20-30之间
- [ ] 高收益债券利差5%-8%

#### 3. Risk-Off 制度
**市场特征**:
- 经济放缓或衰退，GDP增长<1%
- 通胀高企或通缩风险
- 企业盈利下降
- 标普500指数跌破关键支撑位
- VIX指数>30（高波动）
- 高收益债券利差扩大

**操作策略**:
- 建议仓位：20%-40%
- 板块选择：防御性资产（公用事业、必需消费）
- 行业偏好：公用事业、必需消费、电信
- 考虑配置：国债、黄金等避险资产

**信号判断**（满足3个以上）:
- [ ] GDP季度增长率<1%
- [ ] CPI通胀>4%或<1%
- [ ] 标普500指数跌破200日均线
- [ ] VIX>30
- [ ] 高收益债券利差>8%

### 宏观指标监控
- **GDP增长率**：季度数据
- **CPI通胀率**：月度数据
- **失业率**：月度数据
- **利率政策**：美联储议息会议
- **收益率曲线**：10年期与2年期国债利差
- **VIX恐慌指数**：实时数据

- [ ] 记录标普500、纳指、道指收盘价和涨跌幅

- [ ] 制定投资组合调整计划
